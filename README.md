# Monostone — AI Smart Ring · iOS Prototype

Static HTML prototype of the Monostone iOS onboarding + main app flow, built from the 10-screen wireframe.

## Screens

1. Splash
2. Problem + Login
3. How It Works
4. Context Connect
5. Ring Pairing
6. Home Dashboard
7. Listening
8. Processing
9. Result (Task page)
10. Memory / Knowledge Graph

## Run locally

Just open `index.html` in a browser. No build step.

## Deploy

Deployed on Vercel. See repo for the live URL.
